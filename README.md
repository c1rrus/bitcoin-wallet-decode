# Bitcoin Wallet Decoder

Tool to export private keys from [Android Bitcoin Wallet](https://github.com/bitcoin-wallet/bitcoin-wallet) back-up.

It basically create a Docker image that contains all the necessary tools, as described in the [Recovering Bitcoins](https://github.com/bitcoin-wallet/bitcoin-wallet/blob/master/wallet/README.recover.md) instructions and also [this guide on the Bitcoin Talk forums](https://bitcointalk.org/index.php?topic=2061691.0).
